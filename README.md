## 실제 이미지 교체
`assets/screenshots/screenshot-1.svg`부터 `screenshot-5.svg`까지를 실제 App Store 스크린샷으로 교체할 수 있습니다. PNG로 바꿀 경우 `index.html`의 파일 확장자도 함께 수정합니다.

- 공식 문의 이메일

- Open Graph 공유 이미지
- Privacy Policy와 Terms 최종 검토
